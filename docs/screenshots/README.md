# Screenshots

Replace these placeholders with real captures. To take them:

1. Open VibeRes (click menu bar icon).
2. Press `⌘⇧4`, then `Space`, then click the popover window.
3. Save as one of:
   - `root.png` — root display list with profile pills
   - `detail.png` — the per-display resolution detail screen
   - `save.png` — the inline "Save profile" form expanded
